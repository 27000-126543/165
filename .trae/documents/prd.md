## 1. 产品概述

智慧矿山安全监测、生产调度与设备全生命周期管理平台，面向煤矿/金属矿等大型矿山企业，集成安全准入、人员定位、生产调度、设备管理、环境监测、质量检测、财务分析等核心业务，实现矿山运营的数字化、智能化和可视化管控。

- 解决矿山安全管控分散、生产调度低效、设备维护滞后等痛点，降低事故率、提升产能利用率
- 目标用户：矿工、调度员、设备管理员、安全员、质量检测员、财务人员、管理层

## 2. 核心功能

| 设备管理员 | 管理员录入 | 设备巡检、振动温度数据上传、维修工单、配件库存 |

### 2.2 功能模块

1. **综合驾驶舱（首页）**：全局数据概览、实时告警摘要、关键指标仪表盘、快速导航
2. **安全准入与人员定位**：矿工健康验证、培训记录核查、准入权限判定、井下实时定位、危险区域报警、救援指令推送
3. **生产调度中心**：采掘面实时产量、运输车辆位置、铲运车路线优化、每日生产任务单生成
4. **设备全生命周期**：设备台账、扫码巡检、振动温度数据采集、故障预测、维修工单、配件库存推荐
5. **环境监测与应急**：瓦斯/粉尘浓度实时监控、阈值设定、超标自动报警、应急广播、人员撤离通知、事故报告
6. **质量检测**：矿石样本数据上传、品位等级自动判定、选矿流程参数调整
7. **财务分析**：班组产量统计、维修成本汇总、能耗数据分析、综合利润报表、管理层推送
8. **消息中心**：入井通知、预警推送、故障告警、质量异常、调度变更、凭证下载

## 3. 核心流程

### 3.1 矿工准入与安全监控流程
矿工下井前系统自动核查健康数据与培训记录，合格则放行，不合格则拒绝准入并推送提醒。下井后定位卡实时追踪，进入危险区域立即声光报警并推送救援指令。

```mermaid
flowchart TD
    "矿工发起入井请求" --> "系统核查健康数据"
    "系统核查健康数据" --> "系统核查培训记录"
    "系统核查培训记录" --> "是否全部合格?"
    "是否全部合格?" -->|是| "准入通过，推送通知"
    "是否全部合格?" -->|否| "准入拒绝，推送提醒"
    "准入通过，推送通知" --> "下井作业"
    "下井作业" --> "定位卡实时追踪"
    "定位卡实时追踪" --> "是否进入危险区域?"
    "是否进入危险区域?" -->|否| "继续追踪"
    "是否进入危险区域?" -->|是| "声光报警+推送救援指令"
```

### 3.2 生产调度流程
调度员查看采掘面产量与车辆位置，系统根据矿石品位优化铲运车路线，自动生成每日任务单。

```mermaid
flowchart TD
    "调度员查看采掘面产量" --> "系统分析矿石品位数据"
    "系统分析矿石品位数据" --> "自动优化铲运车路线"
    "自动优化铲运车路线" --> "减少空驶率"
    "减少空驶率" --> "生成每日生产任务单"
    "生成每日生产任务单" --> "调度员确认下发"
```

### 3.3 设备全生命周期管理流程
设备管理员扫码巡检上传数据，系统预测剩余寿命，提前生成维修工单并推荐配件。

```mermaid
flowchart TD
    "设备管理员扫码巡检" --> "上传振动/温度数据"
    "上传振动/温度数据" --> "系统分析历史故障模式"
    "系统分析历史故障模式" --> "预测设备剩余寿命"
    "预测设备剩余寿命" --> "是否即将故障?"
    "是否即将故障?" -->|是| "自动生成维修工单"
    "是否即将故障?" -->|否| "继续监控"
    "自动生成维修工单" --> "推荐配件库存"
    "推荐配件库存" --> "通知维修人员"
```

### 3.4 环境应急流程
安全员设定阈值，系统连续监测，超标自动触发应急广播和撤离通知，生成事故报告。

```mermaid
flowchart TD
    "安全员设定瓦斯/粉尘阈值" --> "系统实时监测浓度"
    "系统实时监测浓度" --> "是否连续超标?"

    "是否连续超标?" -->|是| "触发应急广播"
    "触发应急广播" --> "推送人员撤离通知"
    "推送人员撤离通知" --> "生成事故报告"
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深蓝黑（#0A1628）作为背景底色，象征矿山深邃与科技感；亮青色（#00F0FF）作为主要强调色，代表数据与智能；琥珀色（#FFB800）作为告警色；红色（#FF3B3B）作为危险色
- **辅助色**：深灰蓝（#1A2A42）卡片背景、翡翠绿（#00D68F）正常状态色、钢蓝（#4A7BCC）辅助色
- **按钮风格**：圆角矩形（8px），主按钮亮青渐变，次按钮边框式，告警按钮琥珀色
- **字体**：标题使用 DIN Alternate（数字仪表感），正文使用 PingFang SC / Noto Sans SC
- **布局风格**：左侧固定导航栏 + 顶部信息栏 + 主内容区，卡片式布局，数据可视化密集展示
- **图标风格**：线性图标（2px线宽），科技感强，搭配微发光效果
- **整体风格**：工业科技仪表盘（Industrial Sci-Fi Dashboard），深色主题，数据驱动，带有微光特效和呼吸动画

- 桌面优先（1920×1080），适配1440px和1280px
- 平板端（1024px）侧边栏折叠为图标模式
- 移动端（768px以下）底部导航+简化卡片布局
